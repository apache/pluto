Getting Started with Pluto

It is expected that the user is familiar with both the Apache Maven project tool and the Apache Ant build tool.

Requirements:

  • Ant 1.5 or higher
  • Maven 1.0-beta-10 or higher
  • Java 1.3 or higher
  • Servlet 2.3:
    Tomcat 4.1.18-LE w/JDK 1.4
    or
    Tomcat 4.1.24 w/ JDK 1.3

1. Get Maven Ready

If you have not already done so, download and install Maven.

2. build.properties settings:

Please set these values in your ${USER_HOME}/build.properties file:

property example what is it?
tomcat.home.pluto /tomcat location of the Tomcat install where you want to deploy Pluto

2. Building and deploying

  • change your directory to root of the Pluto project.
  • maven fullDeployment
  • This will take care of everything for you including copying shared jars, and deploy the base Pluto portal along with the Portlet Test Suite

3. Start Tomcat

Run Tomcat, go to http://localhost:8080/pluto/portal.
