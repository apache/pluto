Page Registry Portlet Help

This portlet is a view of data contained in the pageregistry.xml file located in /webapps/pluto/WEB-INF/data. This file is used to define a portlet's page layout.

View Mode

Each row represents the layout of a page or portlet application in Pluto. In pageregistry.xml this is manifested by a fragment element with a type attribute of 'page'. The Page column in the table is the value of the title element in pageregistry.xml, which is also used as the navigation link in Pluto. The number of rows and columns in the layout also shown in the table.

This portlet is a simplified view of the pageregistry.xml file. For more details on the file's structure see http://portals.apache.org/pluto/faq.html#portal
